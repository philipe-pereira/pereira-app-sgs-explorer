# SGS Explorer

Aplicativo para consumo e exploração de dados do SGS (Sistema Gerenciador de Séries Temporais) do Banco Central do Brasil.

## Tipo
- **Aplicação** (`pereira-app-*`)

## Build
```bash
mvn -DskipTests clean package
```

## Main class
`IndEco`

## Observação
Nome legado: **IndicadoresEconomicos**.
